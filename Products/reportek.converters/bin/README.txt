============================
Python GMLConverters package
============================

About
-----

This is the GMLConverters package, a collection of Python modules
implementing various converters from/into GML

Dependencies
------------

- Python 2.2.0 or later
- Shapelib 1.2.9 (http://shapelib.maptools.org/)
- Pyshapelib 0.3 (http://ftp.intevation.de/users/bh/pyshapelib/)
- Python Imaging Library (PIL) 1.1.5 (http://www.pythonware.com/products/pil/)
